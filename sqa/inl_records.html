INL Software Quality Records

Several software quality-related documents and records are only available with access to the INL network. Instructions on how to access these documents are provided below. These are stored in the same way for each MOOSE-based application, so where appropriate, use the name of the actual application of interest when searching.

warningwarning:INL Network Required

To access the following documents you must be on the INL network.

Software Quality Assurance Plan (SQAP)

The SQAP used is titled the "Software Quality Assurance Plan for MOOSE and MOOSE-Based Applications" (PLN-4005). This plan lays out the specifics of how MOOSE and MOOSE-based applications ensure that life-cycle activities are planned and performed in a traceable and orderly manner. It includes the requirements for Configuration Management Plan (CMP), Verification and Validation Plan (VVP), and Software Quality Plan (SQP). PLN-4005 can be accessed from the INL network as follows.

  1. From the Nucleus Home page select A-Z

  2. From the alphabetical list select EDMS

  3. Enter PLN-4005 in the "Document Id" field and select Submit.

  4. Select the Doc Viewer down arrow and then select the Record File pdf.

Enterprise Architecture (EA) Record

The EA record can be accessed from the INL network as follows.

  1. From the Nucleus Home page select A-Z

  2. From the alphabetical list select CTM—Capabilities & Technology Management

  3. Begin typing the name of the application in the search box in the upper right corner of the screen

  4. When the search is complete, click on the ellipses (three dots) to the left of the application name and then select View Details

Safety Software Determination (SSD)

The SSD can be accessed from the INL network as follows.

  1. From the EA Details screen for the application (refer to EA Record), select the General tab

  2. Scroll to the bottom of the screen and in the Related Artifacts section select Safety Software Determination

Quality Level Determination (QLD)

The QLD can be accessed from the INL network as follows.

  1. From the EA Details screen for the application (refer to EA Record), select the General tab

  2. Scroll to the bottom of the screen and in the Related Artifacts section select Quality Level Determination

Configuration Management Plan (CMP)

The CMP is included in the SQAP, see SQAP.

Verification and Validation Plan (VVP)

The VVP is included in the SQAP, see SQAP.

Software Quality Plan (SQP)

The SQP is included in the SQAP, see SQAP.

Asset Management Plan (AMP)

In accordance with INL Laboratory Wide Procedure (LWP) "Managing Information Technology Assets" (LWP-13620), an Information Technology (IT) asset maintenance plan, or equivalent, must be created by the end user to specify how the software application will be managed after it is deployed for use, see the INL template, "IT Asset Maintenance Plan Template" (TEM-215) for further information.

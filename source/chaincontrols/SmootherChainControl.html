SmootherChainControl

This ChainControl smooths an input signal using a moving average. The most recent values are used to compute the average of the input values :

where represents the time index.

The resulting value is named <control>:value, where <control> is the user-given name of the SmootherChainControl. The number of points to average, , is provided with "n_points".

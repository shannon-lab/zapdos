AxisymmetricCenterlineAverageValue

Computes the average value of a variable on a sideset located along the centerline of an axisymmetric model.

Description

'AxisymmetricCenterlineAverageValue' is a specialized case of the SideAverageValue postprocessor for cases in which the sideset on which the average value is being calculated is along the centerline axis in axisymmetric simulations.

Example Input syntax

[Postprocessors<<<{"href": "../../syntax/Postprocessors/index.html"}>>>]
  [average]
    type = AxisymmetricCenterlineAverageValue<<<{"description": "Computes the average value of a variable on a sideset located along the centerline of an axisymmetric model.", "href": "AxisymmetricCenterlineAverageValue.html"}>>>
    boundary<<<{"description": "The list of boundary IDs from the mesh where this object applies"}>>> = left
    variable<<<{"description": "The name of the variable which this postprocessor integrates"}>>> = u
  []
[]

TimestepSize

Reports the timestep size

Description

TimestepSize reports the timestep size.

Example Input Syntax

[Postprocessors<<<{"href": "../../syntax/Postprocessors/index.html"}>>>]
  [./_dt]
    type = TimestepSize<<<{"description": "Reports the timestep size", "href": "TimestepSize.html"}>>>
  [../]
[]
(moose/test/tests/time_steppers/iteration_adaptive/adapt_tstep_grow_init_dt.i)

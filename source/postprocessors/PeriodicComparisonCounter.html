PeriodicComparisonCounter

buildconstruction:Undocumented Class

The PeriodicComparisonCounter has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Compares two Postprocessors or values and keeps countof how many cycles in a row that comparision is true.

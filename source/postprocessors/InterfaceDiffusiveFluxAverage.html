InterfaceDiffusiveFluxAverage

Computes the diffusive flux on the interface.

The diffusive flux average is defined as with the interface, the diffusion coefficient, the variable, the normal to the interface and the surface area of the interface.

commentnote

For finite volume variables, this postprocessor computes the diffusive flux using a two point gradient. This is only accurate if the (interface) kernel is also computing gradients this way. Currently, only FVDiffusionInterface is computing gradients this way.

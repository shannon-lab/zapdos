RelativeElementL2Difference

buildconstruction:Undocumented Class

The RelativeElementL2Difference has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Computes the element-wise relative L2 difference between the current variable and a coupled variable: i.e. ||u-v||/||u||

MultiPeriodAverager

MultiPeriodAverager calculates the the period average of values of periodic fuctions at the end of each period, such that: Where is the value of MultiPeriodAverager, is the number of periods to average over, and is the value of the periodic value at the end of a period. Until the number of periods over which you are averaging has passed the value of the Postprocessor will reflect the previously calculated average.

warningwarning

Before the first average is calculated the value of this post processor will be 0.

commentnote:Proper Usage Hint

MultiPeriodAverager is intended to be used with the PeriodicTimeIntegratedPostprocessor.

Example Input File Syntax

[Postprocessors]
  [a]
    type = ElementIntegralVariablePostprocessor
    variable = v
    execute_on = 'initial timestep_end'
  []

  [periodic]
    type = PeriodicTimeIntegratedPostprocessor
    value = a
    execute_on = 'initial timestep_end'
  []

  [multi_period]
    type = MultiPeriodAverager
    value = periodic
    number_of_periods = 2
  []
[]
(test/tests/postprocessors/multi_period_averaging/multi_period_averager.i)

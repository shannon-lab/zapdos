SideIntegralFunctorPostprocessor

Computes a surface integral of the specified functor, using the single-sided face argument, which usually means that the functor will be evaluated from a single side of the surface, not interpolated between both sides.

The SideIntegralFunctorPostprocessor is also an intermediate base class that should be derived from for any calculation involving the integral of a variable quantity over a side.

commentnote

Functors are currently restricted to finite volume variables, functor material properties and functions.

Input Parameters

  • boundaryThe list of boundary IDs from the mesh where this boundary condition applies

    C++ Type:std::vector<BoundaryName>

    Controllable:No

    Description:The list of boundary IDs from the mesh where this boundary condition applies

  • functorThe name of the functor that this postprocessor integrates

    C++ Type:MooseFunctorName

    Controllable:No

    Description:The name of the functor that this postprocessor integrates

    Default:TIMESTEP_END

    C++ Type:ExecFlagEnum

    Options:NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, FINAL, CUSTOM, ALWAYS, TRANSFER

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed, the available options include NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, FINAL, CUSTOM, ALWAYS.

    C++ Type:MooseFunctorName

    Controllable:No

    C++ Type:MaterialPropertyName

    Controllable:No

    Description:An optional suffix parameter that can be appended to any attempt to retrieve/get material properties. The suffix will be prepended with a '_' character.

    Default:False

    C++ Type:bool

    Description:If the functor (and the prefactor) is only defined only along part of the sideset, allows to skip the parts where it is not defined. Please keep in mind that if the sideset is defined with the wrong normal, this may allow to skip the entire integral.

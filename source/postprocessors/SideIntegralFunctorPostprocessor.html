SideIntegralFunctorPostprocessor

Computes a surface integral of the specified functor, using the single-sided face argument, which usually means that the functor will be evaluated from a single side of the surface, not interpolated between both sides.

The SideIntegralFunctorPostprocessor is also an intermediate base class that should be derived from for any calculation involving the integral of a functor quantity over a side.

The "functor_argument" parameter lets the user select between face integration and element side quadrature point integration. Face integration is more appropriate for finite volume variables, and quadrature point integration is more appropriate for finite element variables.

Input Parameters

  • boundaryThe list of boundary IDs from the mesh where this object applies

    C++ Type:std::vector<BoundaryName>

    Controllable:No

    Description:The list of boundary IDs from the mesh where this object applies

  • functorThe name of the functor that this postprocessor integrates. A functor is any of the following: a variable, a functor material property, a function, a postprocessor or a number.

    C++ Type:MooseFunctorName

    Unit:(no unit assumed)

    Controllable:No

    Description:The name of the functor that this postprocessor integrates. A functor is any of the following: a variable, a functor material property, a function, a postprocessor or a number.

    Controllable:No

    Description:Location to evaluate the functors at

    Default:1

    C++ Type:MooseFunctorName

    Unit:(no unit assumed)

    Controllable:No

    Description:Factor multiplying the integrand. A functor is any of the following: a variable, a functor material property, a function, a postprocessor or a number.

  • restrict_to_functors_domainFalseIf the functor (and the prefactor) is only defined only along part of the sideset, allows to skip the parts where it is not defined. Please keep in mind that if the sideset is defined with the wrong normal, this may allow to skip the entire integral.

    Default:False

    C++ Type:bool

    Controllable:No

    Description:If the functor (and the prefactor) is only defined only along part of the sideset, allows to skip the parts where it is not defined. Please keep in mind that if the sideset is defined with the wrong normal, this may allow to skip the entire integral.

NodalL2Norm

Computes the nodal L2-norm of the coupled variable, which is defined by summing the square of its value at every node and taking the square root.

Example input syntax

In this example, the L2 norm of the variable saved is computed on block 0 at the end of every time step using a NodalL2Norm postprocessor.

[Postprocessors<<<{"href": "../../syntax/Postprocessors/index.html"}>>>]
  [./left_flux]
    type = NodalSum<<<{"description": "Computes the sum of all of the nodal values of the specified variable. Note: This object sets the default \"unique_node_execute\" flag to true to avoid double counting nodes between shared blocks.", "href": "NodalSum.html"}>>>
    variable<<<{"description": "The name of the variable that this postprocessor operates on"}>>> = saved
    boundary<<<{"description": "The list of boundaries (ids or names) from the mesh where this object applies"}>>> = 1
  [../]
  [./saved_norm]
    type = NodalL2Norm<<<{"description": "Computes the nodal L2-norm of the coupled variable, which is defined by summing the square of its value at every node and taking the square root.", "href": "NodalL2Norm.html"}>>>
    variable<<<{"description": "The name of the variable that this postprocessor operates on"}>>> = saved
    execute_on<<<{"description": "The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html."}>>> = timestep_end
    block<<<{"description": "The list of blocks (ids or names) that this object will be applied"}>>> = 0
  [../]
  [./saved_dirichlet_norm]
    type = NodalL2Norm<<<{"description": "Computes the nodal L2-norm of the coupled variable, which is defined by summing the square of its value at every node and taking the square root.", "href": "NodalL2Norm.html"}>>>
    variable<<<{"description": "The name of the variable that this postprocessor operates on"}>>> = saved_dirichlet
    execute_on<<<{"description": "The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html."}>>> = timestep_end
    block<<<{"description": "The list of blocks (ids or names) that this object will be applied"}>>> = 0
  [../]
[]
(moose/test/tests/misc/save_in/diag_save_in_soln_var_err_test.i)

    boundary = left
    value = 0
    save_in = saved_dirichlet

  [./nbc]
    type = NeumannBC

    boundary = right
    value = 1
    save_in = 'bc_saved accumulated'
    diag_save_in = bc_diag_saved

    type = NodalSum

ElementL2Norm

Computes a volume integral of the specified variable

The element norm is defined as:

The element norm of a variable is computed as:

The term accounts for the geometry/volume of the element, the for the weight in the quadrature, for an eventual coordinate change (cylindrical, spherical) and is the value of the variable at the quadrature point.

Example input syntax

In this example, we compute the L2 norm for a variety of auxiliary variable types using ElementL2Norm postprocessors.

[Postprocessors<<<{"href": "../../syntax/Postprocessors/index.html"}>>>]
  [./int2_u]
    type = ElementL2Norm<<<{"description": "Computes a volume integral of the specified variable", "href": "ElementL2Norm.html"}>>>
    variable<<<{"description": "The name of the variable that this object operates on"}>>> = u
    execute_on<<<{"description": "The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html."}>>> = 'initial timestep_end'

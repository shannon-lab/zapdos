ElementVectorL2Error

Computes the Vector L2 difference of up to three variables simultaneously (normally x, y, z)

Overview

Example Input File Syntax

Input Parameters

  • function_xThe analytic solution to compare against

    C++ Type:FunctionName

    Options:

    Description:The analytic solution to compare against

  • var_xThe FE solution in x direction

    C++ Type:std::vector<VariableName>

    Options:

    Description:The FE solution in x direction

Required Parameters

  • blockThe list of blocks (ids or names) that this object will be applied

    C++ Type:std::vector<SubdomainName>

    Options:

    Description:The list of blocks (ids or names) that this object will be applied

  • execute_onTIMESTEP_ENDThe list of flag(s) indicating when this object should be executed, the available options include NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, FINAL, CUSTOM.

    Default:TIMESTEP_END

    C++ Type:ExecFlagEnum

    Options:NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, FINAL, CUSTOM, TRANSFER

    Description:The list of flag(s) indicating when this object should be executed, the available options include NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, FINAL, CUSTOM.

  • function_y0The analytic solution to compare against

    Default:0

    C++ Type:FunctionName

    Options:

    Description:The analytic solution to compare against

  • function_z0The analytic solution to compare against

    Default:0

    C++ Type:FunctionName

    Options:

    Description:The analytic solution to compare against

    C++ Type:MaterialPropertyName

    Options:

    Description:An optional suffix parameter that can be appended to any attempt to retrieve/get material properties. The suffix will be prepended with a '_' character.

  • var_yThe FE solution in y direction

    C++ Type:std::vector<VariableName>

    Options:

    Description:The FE solution in y direction

  • var_zThe FE solution in z direction

    C++ Type:std::vector<VariableName>

    Options:

    Description:The FE solution in z direction

AddPeriodicControllers

This Action automatically adds multiple 'TimePeriod' controllers for the purpose of enabling and disabling multiple objects during multiple cycles. (generally for periodic accelerations)

Overview

The PeriodicControllers object automates the supply of TimePeriod controls to enable or disable objects at the beginning, during, or at the end of periodic cycles.

Example Input File Syntax

[PeriodicControllers<<<{"href": "../../syntax/PeriodicControllers/index.html"}>>>]
  [Shooting]
    enable_at_cycle_start<<<{"description": "A list of objects names to enable at the start of the cycle."}>>> = '*::Ar*S_for_Shooting'

    enable_during_cycle<<<{"description": "A list of objects names to enable during the cycle."}>>> = '*::SM_Ar*_time_deriv *::SM_Ar*_diffusion *::SM_Ar*_stepwise_ionization
                           *::SM_Ar*_collisions *::SM_Ar*_quenching *::SM_Ar*_pooling
                           *::SM_Ar*_2B_quenching *::SM_Ar*_3B_quenching'

    enable_at_cycle_end<<<{"description": "A list of objects names to enable at the end of the cycle."}>>> = 'MultiApps::Shooting
                           *::SM_Ar*Reset_to_Shooting *::Ar*_to_Shooting
                           *::Ar*S_to_Shooting *::Ar*T_to_Shooting
                           *::SMDeriv_to_Shooting *::Ar*New_from_Shooting
                           *::SM_Ar*Reset_from_Shooting *::Ar*Relative_Diff'
    cycle_frequency<<<{"description": "The cycle's frequency in Hz"}>>> = 13.56e6
    #starting_cycle = 25
    #cycles_between_controls = 25
    starting_cycle<<<{"description": "The number of the cycles before starting the control scheme"}>>> = 50
    cycles_between_controls<<<{"description": "The number of cycles between controllers(e.g. the number of cycles between accelerations)"}>>> = 50
    cycles_per_controls<<<{"description": "The number of cycles that occur during the controllers(e.g. the number of cycles during the accelerations)"}>>> = 1
    num_controller_set<<<{"description": "The number of controller sets(e.g. the number of accelerations)"}>>> = 2
    name<<<{"description": "Custom name for the 'PeriodicControllers'."}>>> = Shooting
  []
[]

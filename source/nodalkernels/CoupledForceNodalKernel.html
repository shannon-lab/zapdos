CoupledForceNodalKernel

Adds a force proportional to the value of the coupled variable

Description

The CoupledForceNodalKernel applies either a sink or source term proportional to the value of a coupled variable v. An optional parameter coef determines whether the term is a source or a sink and the proprtionality between the coupled variable and the applied force. When coef is positive (the default value is 1), then the term is a source.

Example Syntax

[NodalKernels]
  [positive_constraint]
    type = LowerBoundNodalKernel
    variable = lm
    v = u
    exclude_boundaries = 'left right'
  []
  [forces]
    type = CoupledForceNodalKernel
    variable = u
    v = lm
  []
[]
(moose/test/tests/nodalkernels/constraint_enforcement/lower-bound.i)

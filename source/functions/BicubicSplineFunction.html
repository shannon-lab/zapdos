BicubicSplineFunction

Define a bicubic spline function from interpolated data defined by input parameters.

The BicubicSplineFunction defines a 2D spline shape, which can be evaluated everywhere in the domain by translation. The 2D plane for defining the spline is set by specifying the normal parameter.

The spline is uniquely defined by:

  • its values the 2D plane at the (x1, x2) points, given by the y parameter. The points form a 2D grid, which each x1 being the abscissa for a line in this grid, with points at each x2 specified

  • its derivatives along x1 and x2 at the points on each extremity, given by yx11, yx1n, yx21, yx2n

  • a functional form for the derivative along both directions, given by yx1 and yx2

From this information the bicubic spline is automatically generated. Both the first and second order derivatives of the spline are defined.

Example input syntax

In this example, we define a bicubic spline from a list of points and derivatives. The z normal is assumed by default and the bicubic spline is defined in the XY plane. The grid for the points has 3 points along the x direction and 4 points along the y direction.

[Functions<<<{"href": "../../syntax/Functions/index.html"}>>>]
  [./yx1]
    type = ParsedFunction<<<{"description": "Function created by parsing a string", "href": "MooseParsedFunction.html"}>>>
    expression<<<{"description": "The user defined function."}>>> = '3*x^2'
  [../]
  [./yx2]
    type = ParsedFunction<<<{"description": "Function created by parsing a string", "href": "MooseParsedFunction.html"}>>>
    expression<<<{"description": "The user defined function."}>>> = '6*y^2'
  [../]
  [./spline_fn]
    type = BicubicSplineFunction<<<{"description": "Define a bicubic spline function from interpolated data defined by input parameters.", "href": "BicubicSplineFunction.html"}>>>
    x1<<<{"description": "The first independent coordinate."}>>> = '0 2 4'
    x2<<<{"description": "The second independent coordinate."}>>> = '0 2 4 6'
    y<<<{"description": "The dependent values"}>>> = '0 16 128 432 8 24 136 440 64 80 192 496'
    yx11<<<{"description": "The values of the derivative wrt x1 on the lower interpolation grid points."}>>> = '0 0 0 0'
    yx1n<<<{"description": "The values of the derivative wrt x1 on the upper interpolation grid points."}>>> = '48 48 48 48'
    yx21<<<{"description": "The values of the derivative wrt x2 on the lower interpolation grid points."}>>> = '0 0 0'
    yx2n<<<{"description": "The values of the derivative wrt x2 on the upper interpolation grid points."}>>> = '216 216 216'
    yx1<<<{"description": "The functional form of the derivative with respect to x1."}>>> = 'yx1'
    yx2<<<{"description": "The functional form of the derivative with respect to x2."}>>> = 'yx2'
  [../]
  [./u_func]
    type = ParsedFunction<<<{"description": "Function created by parsing a string", "href": "MooseParsedFunction.html"}>>>
    expression<<<{"description": "The user defined function."}>>> = 'x^3 + 2*y^3'
  [../]
  [./u2_forcing_func]
    type = ParsedFunction<<<{"description": "Function created by parsing a string", "href": "MooseParsedFunction.html"}>>>
    expression<<<{"description": "The user defined function."}>>> = '-6*x - 12*y'
  [../]
[]

ADFluxFromGradientMaterial

Computes a flux vector material property based on the gradient of a coupled variable and a scalar diffusivity.

Overview

ADFluxFromGradientMaterial computes a vector-valued flux based on the gradient of a coupled variable and a scalar diffusivity material property.

It implements the relation:

[ \mathbf{J} = -D \nabla u ]

where:

  • \( \mathbf{J} \) is the flux (vector)

  • \( D \) is the scalar diffusivity (material property)

  • \( u \) is the coupled variable

This flux can be used with the ADFluxDivergence kernel to model diffusion-like processes.

Example Input File

[Materials<<<{"href": "../../syntax/Materials/index.html"}>>>]
  [flux]
    type = ADFluxFromGradientMaterial<<<{"description": "Computes a flux vector material property based on the gradient of a coupled variable and a scalar diffusivity.", "href": "ADFluxFromGradientMaterial.html"}>>>
    flux<<<{"description": "Name of the flux material property"}>>> = flux
    u<<<{"description": "Variable used to compute the flux"}>>> = u
    diffusivity<<<{"description": "The diffusivity material property name"}>>> = diffusivity
  []
[]

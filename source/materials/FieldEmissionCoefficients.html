FieldEmissionCoefficients

The material coefficients for field emission

Overview

FieldEmissionCoefficients defines the coefficients needed for the field emission boundary conditions (i.e., FieldEmissionBC and SchottkyEmissionBC). The following are the properties and naming scheme that FieldEmissionCoefficients provides:

  • the work function, labeled as work_function,

  • the field enhancement factor, labeled as field_enhancement, and

  • the Richardson coefficient, labeled as Richardson_coefficient.

Example Input File Syntax

[Materials<<<{"href": "../../syntax/Materials/index.html"}>>>]
  [gas_block_field_emission]
    type = FieldEmissionCoefficients<<<{"description": "The material coefficients for field emission", "href": "FieldEmissionCoefficients.html"}>>>
    user_work_function<<<{"description": "The work function."}>>> = 4.55 # eV
    user_field_enhancement<<<{"description": "The field enhancement factor."}>>> = 55
    user_Richardson_coefficient<<<{"description": "The Richardson coefficient."}>>> = 80E4
    block<<<{"description": "The list of blocks (ids or names) that this object will be applied"}>>> = 0
  []

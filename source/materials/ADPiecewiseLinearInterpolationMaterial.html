ADPiecewiseLinearInterpolationMaterial

Compute a property using a piecewise linear interpolation to define its dependence on a variable

This model computes a material property using a piecewise linear interpolation to define the dependence of the property on a specified variable. The property is computed as using the piecewise linear interpolation, where the coupled variable provides the value of at each location where this is evaluated.

Input Parameters

  • propertyThe name of the property this material will compute

    C++ Type:std::string

    Unit:(no unit assumed)

    Controllable:No

    Description:The name of the property this material will compute

  • variableThe name of the variable whose value is used as the abscissa in the interpolation

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:The name of the variable whose value is used as the abscissa in the interpolation

    C++ Type:std::vector<SubdomainName>

    Unit:(no unit assumed)

    Controllable:No

    Description:The list of blocks (ids or names) that this object will be applied

  • boundaryThe list of boundaries (ids or names) from the mesh where this object applies

    Description:The list of boundaries (ids or names) from the mesh where this object applies

    Default:True

    C++ Type:bool

    Unit:(no unit assumed)

    Controllable:No

    Description:When false, MOOSE will not call compute methods on this material. The user must call computeProperties() after retrieving the MaterialBase via MaterialBasePropertyInterface::getMaterialBase(). Non-computed MaterialBases are not sorted for dependencies.

Evaluations on element qps will be skipped

    Unit:(no unit assumed)

Evaluations on element qps will be skipped

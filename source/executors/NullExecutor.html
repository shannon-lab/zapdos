NullExecutor

Dummy executor that does nothing. Useful for testing among other things.

The NullExecutor is an object used for testing and for temporarily replacing a different executor when doing lazy binding. It should not be created by users.

Input Parameters

  • begin_exec_flagNONEexec flag associated with the beginning of this executor

    Default:NONE

    C++ Type:MooseEnumItem

    Controllable:No

    Description:exec flag associated with the beginning of this executor

  • end_exec_flagNONEexec flag associated with the end of this executor

    Default:NONE

    C++ Type:MooseEnumItem

    Controllable:No

    Description:exec flag associated with the end of this executor

  • verboseFalseSet to true to print additional information

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Set to true to print additional information

DGCoupledAdvection

buildconstruction:Undocumented Class

The DGCoupledAdvection has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

DG upwinding for convection on Navier-Stokes velocity variables

Overview

    Controllable:No

    Description:The name of the variable that this residual object operates on

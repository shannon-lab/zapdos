DiffusionHDGDirichletBC

This class has some conceptual similarities to FunctionDirichletBC, however, the Dirichlet condition described by "functor" is imposed weakly for a hybridized discontinuous Galerkin discretization instead of strongly for a continuous Galerkin discretization. This boundary condition should be paired with a DiffusionHDGKernel.

    Unit:(no unit assumed)

  • grad_uThe gradient of the diffusing specie concentration

    C++ Type:AuxVariableName

    Unit:(no unit assumed)

    Controllable:No

    Description:The gradient of the diffusing specie concentration

    C++ Type:AuxVariableName

    Unit:(no unit assumed)

    Description:The name of the variable that this residual object operates on

    C++ Type:double

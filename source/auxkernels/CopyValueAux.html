CopyValueAux

Returns the specified variable as an auxiliary variable with a simple copy of the variable values.

The CopyValueAux can be used to make a copy of a field variable, for example to lag them in certain numerical schemes. Many AuxKernels can use variables as arguments, without modifying them, and store the result in a separate variable. The use of a CopyValueAux can often be avoided for that reason.

Input Parameters

  • sourceVariable to take the value of.

    C++ Type:std::vector<VariableName>

    Unit:(no unit assumed)

    Controllable:No

    Description:Variable to take the value of.

  • variableThe name of the variable that this object applies to

    C++ Type:AuxVariableName

    Unit:(no unit assumed)

    Controllable:No

    Description:The name of the variable that this object applies to

Setting this to false will allow contribution to a single element's elemental value(s) from multiple boundary sides on the same element (example: when the restricted boundary exists on two or more sides of an element, such as at a corner of a mesh

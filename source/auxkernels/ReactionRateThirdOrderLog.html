ReactionRateThirdOrderLog

buildconstruction:Undocumented Class

The ReactionRateThirdOrderLog has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Reaction rate for three body collisions. User can pass choice of elastic, excitation, or ionization reaction rate coefficients. Automatically added if track_rates set to true in the Reactions action.

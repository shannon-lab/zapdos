VectorVariableMagnitudeAux

The VectorVariableMagnitudeAux class takes a vector variable, specified through the vector_variable parameter, and generates an auxiliary variable corresponding to the Euclidean norm of the vector variable's components. This object is only meant to be used with LAGRANGE_VEC vector variables, and hence the auxiliary variable should be of type LAGRANGE.

Example Input Syntax

[AuxKernels<<<{"href": "../../syntax/AuxKernels/index.html"}>>>]
  [./calc_u_mag]
    type = VectorVariableMagnitudeAux<<<{"description": "Creates a field consisting of the magnitude of a coupled vector variable.", "href": "VectorVariableMagnitudeAux.html"}>>>
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = u_mag # the auxvariable to compute
    vector_variable<<<{"description": "The variable from which to compute the component"}>>> = u # vector variable to compute from
    execute_on<<<{"description": "The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html."}>>> = 'initial timestep_end'
  [../]
[]

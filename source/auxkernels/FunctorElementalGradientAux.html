FunctorElementalGradientAux

Evaluates the gradient of a functor (variable, function or functor material property) on the current element or quadrature point.

Overview

This AuxKernel outputs a vector AuxVariable given by scaling the gradient of a functor scaled by a material property and/or another functor.

commentnote:Use in Optimization module

This AuxVariable is used to linearize a nonlinear material property in the adjoint equation.

Example Input File Syntax

The use of this AuxKernel to compute the adjoint for a temperature dependent material is shown in this example:

[AuxKernels<<<{"href": "../../syntax/AuxKernels/index.html"}>>>]
  [dDdTgradT]
    type = ADFunctorElementalGradientAux<<<{"description": "Evaluates the gradient of a functor (variable, function or functor material property) on the current element or quadrature point.", "href": "FunctorElementalGradientAux.html"}>>>
    functor<<<{"description": "The functor to evaluate. A functor is any of the following: a variable, a functor material property, a function, a postprocessor or a number."}>>> = forwardT
    variable<<<{"description": "The name of the variable that this object applies to"}>>> = dDdTgradT
    factor_matprop<<<{"description": "A (regular) material property factor to apply on the functor"}>>> = 'dDdT'
  []
[]
(moose/modules/combined/test/tests/optimization/invOpt_nonlinear/adjoint.i)

where the AuxVariable dDdTgradT is defined as a constant monomial vector:

[AuxVariables<<<{"href": "../../syntax/AuxVariables/index.html"}>>>]
  [dDdTgradT]
    order<<<{"description": "Specifies the order of the FE shape function to use for this variable (additional orders not listed are allowed)"}>>> = CONSTANT
    family<<<{"description": "Specifies the family of FE shape functions to use for this variable"}>>> = MONOMIAL_VEC
  []
[]
(moose/modules/combined/test/tests/optimization/invOpt_nonlinear/adjoint.i)

In this example, "factor_matprop" is the tangent modulus of the temperature dependent thermal conductivity. FunctorElementalGradientAux provides an advection velocity in the adjoint problem solved by the LevelSetAdvection kernel. For Hessian based inversion, FunctorElementalGradientAux provides the advection velocity to the ConservativeAdvection kernel.

    Options:NONE, INITIAL, LINEAR, LINEAR_CONVERGENCE, NONLINEAR, NONLINEAR_CONVERGENCE, POSTCHECK, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, MULTIAPP_FIXED_POINT_CONVERGENCE, FINAL, CUSTOM, PRE_DISPLACE

    Description:The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html.

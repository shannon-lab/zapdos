ElementCentroidPositions

Positions of element centroids.

Overview

The ElementCentroidPositions may be block-restricted to limit the gathering of elements to certain subdomains.

commentnote

The ElementCentroidPositions, when used in conjunction with a TransientMultiApp enables creating the same subapps as the CentroidMultiApp, except that it is not limited to transients.

Input Parameters

  • blockThe list of blocks (ids or names) that this object will be applied

    C++ Type:std::vector<SubdomainName>

    Controllable:No

    Description:The list of blocks (ids or names) that this object will be applied

  • initial_positionsPositions at the beginning of the simulation

    C++ Type:PositionsName

    Controllable:No

    Description:Positions at the beginning of the simulation

ExtraIDIntegralReporter

This ExtraIDIntegralReporter source code is to integrate variables based on parsed extra IDs based on reporter system.

Overview

The ExtraIDIntegralReporter object is a reporter object to integrate input variables based on multiple extra element IDs. This object internally utilizes the vector postprocessor ExtraIDIntegralVectorPostprocessor to evaluate the integral values. Refer to ExtraIDIntegralVectorPostprocessor for details.

commentnote:Reporter names

The reporters created by the ExtraIDIntegralReporter are the extra_id_data, holding the "id_name" requested and map from names to indexes into the integrals, and the integrals, holding the integral values.

Example Syntax

[Reporters<<<{"href": "../../syntax/Reporters/index.html"}>>>]
  [extra_id_integral]
    type = ExtraIDIntegralReporter<<<{"description": "This ExtraIDIntegralReporter source code is to integrate variables based on parsed extra IDs based on reporter system.", "href": "ExtraIDIntegralReporter.html"}>>>
    variable<<<{"description": "The names of the variables that this ExtraIDIntegralReporter operates on"}>>> = 'value1'
    id_name<<<{"description": "The list of extra ids name by which to separate integrals."}>>> = 'assembly_id'
  []
[]
(moose/test/tests/reporters/extra_id_integral/extra_id_integral.i)

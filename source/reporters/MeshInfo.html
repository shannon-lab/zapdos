MeshInfo

A Reporter object for tracking mesh information, such as the number of elements, nodes, or degrees-of-freedom. The "items" parameter controls the items computed. MeshInfo will declare reporters for each item requested.

Example Input Syntax

The following input file snippet demonstrates the use of the MeshInfo object.

[Reporters]
  [mesh_info]
    type = MeshInfo
  []
[]
(moose/test/tests/reporters/mesh_info/mesh_info.i)

    Default:TIMESTEP_END

    C++ Type:ExecFlagEnum

    Options:NONE, INITIAL, LINEAR, NONLINEAR, POSTCHECK, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, FINAL, CUSTOM

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed, the available options include NONE, INITIAL, LINEAR, NONLINEAR, POSTCHECK, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, FINAL, CUSTOM.

    Description:An optional suffix parameter that can be appended to any attempt to retrieve/get material properties. The suffix will be prepended with a '_' character.

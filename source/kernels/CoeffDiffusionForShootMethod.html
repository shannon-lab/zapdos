CoeffDiffusionForShootMethod

buildconstruction:Undocumented Class

The CoeffDiffusionForShootMethod has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

The derivative of the generic diffusion term used to calculate the sensitivity value for the shoothing method.(Densities must be in log form)

ADMatReaction

Kernel representing the contribution of the PDE term , where is a reaction rate material property, is a scalar variable (nonlinear or coupled), and whose Jacobian contribution is calculated using automatic differentiation.

Implements where ("reaction_rate") is a reaction rate, is either a coupled variable ("v") or - if not explicitly specified - the non-linear variable the kernel is operating on.

Forward automatic differentiation is used to compute all on and off-diagonal Jacobian contributions.

warningwarning

Note the negative sign, which does not appear in Reaction or CoefReaction.

Example Input File Syntax

[Kernels]
  [reaction]
    type = ADMatReaction
    variable = u
    reaction_rate = rxn_prop
  []
[]
(moose/test/tests/kernels/ad_mat_reaction/ad_mat_reaction.i)

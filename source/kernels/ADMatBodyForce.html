ADMatBodyForce

Description

MatBodyForce implements a force term given via a material property. This kernel leverages the formation in ADBodyForce by multiplying the residual by a material property. All other implementation details are the same as ADBodyForce.

Example Syntax

[Kernels]
  [forcing]
    type = ADMatBodyForce
    variable = u
    material_property = forcing_material
  []
[]
(moose/test/tests/kernels/body_force/ad_mat_forcing_function_test.i)

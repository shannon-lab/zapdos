ShootMethodLog

buildconstruction:Undocumented Class

The ShootMethodLog has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

An acceleration scheme based on the shooting method

Overview

Example Input File Syntax

Input Parameters

  • density_at_end_cycleThe accelerated density at the end of the cycle in log form

    C++ Type:std::vector<VariableName>

    Controllable:No

    Description:The accelerated density at the end of the cycle in log form

  • density_at_start_cycleThe accelerated density at the start of the cycle in log form

    C++ Type:std::vector<VariableName>

    Controllable:No

    Description:The accelerated density at the start of the cycle in log form

  • sensitivity_variableThe variable that represents the sensitivity of accelerationas defined for the shooting method

    Description:The name of the variable that this residual object operates on

    Controllable:No

    Description:A limit of the growth factor(growth_limit = 0.0 means no limit)

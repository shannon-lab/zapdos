ReactionThirdOrderLogForShootMethod

The derivative of a third order reaction term used to calculate the sensitivity variable for the shoothing method. (Densities must be in logarithmic form)

Overview

ReactionThirdOrderLogForShootMethod is the derivative of a third-order reaction term used to calculate the sensitivity variable for the shooting method. For more information on the implementation of the shooting method within Zapdos, please reference ShootMethodLog.

Example Input File Syntax

An example of how to use ReactionThirdOrderLogForShootMethod can be found in the test file Acceleration_By_Shooting_Method.i.

[Kernels<<<{"href": "../../syntax/Kernels/index.html"}>>>]
  [SM_Ar*_3B_quenching]
    type = ReactionThirdOrderLogForShootMethod<<<{"description": "The derivative of a third order reaction term used to calculate the sensitivity variable for the shoothing method. (Densities must be in logarithmic form)", "href": "ReactionThirdOrderLogForShootMethod.html"}>>>
    variable<<<{"description": "The name of the variable that this residual object operates on"}>>> = SM_Ar*
    density<<<{"description": "The accelerated density variable."}>>> = Ar*
    v<<<{"description": "The Second variable that is reacting."}>>> = Ar
    w<<<{"description": "The Third variable that is reacting."}>>> = Ar
    reaction<<<{"description": "The full reaction equation."}>>> = 'Ar* + Ar + Ar -> Ar_2 + Ar'
    coefficient<<<{"description": "The stoichiometric coeffient."}>>> = -1
    enable<<<{"description": "Set the enabled status of the MooseObject."}>>> = false
  []
[]

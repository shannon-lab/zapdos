VectorCoupledTimeDerivative

Time derivative Kernel that acts on a coupled vector variable. Weak form: .

Overview

The VectorCoupledTimeDerivative kernel is very similar to the VectorTimeDerivative kernel with the exception that the time derivative operator is applied to a coupled variable instead of the variable to whom's residual the VectorCoupledTimeDerivative kernel contributes. [Kernels<<<{"href": "../../syntax/Kernels/index.html"}>>>]
  [./time_u]
    type = VectorTimeDerivative<<<{"description": "The time derivative operator with the weak form of $(\\vec{\\psi_i}, \\frac{\\partial \\vec{u_h}}{\\partial t})$.", "href": "VectorTimeDerivative.html"}>>>

  [../]
  [./fn_u]
    type = VectorBodyForce<<<{"description": "Demonstrates the multiple ways that scalar values can be introduced into kernels, e.g. (controllable) constants, functions, and postprocessors. Implements the weak form $(\\vec{\\psi_i}, -\\vec{f})$.", "href": "VectorBodyForce.html"}>>>
    variable<<<{"description": "The name of the variable that this residual object operates on"}>>> = u
    function_x<<<{"description": "A function that describes the x-component of the body force"}>>> = 1
    function_y<<<{"description": "A function that describes the y-component of the body force"}>>> = 1
  [../]
  [./time_v]
    type = VectorCoupledTimeDerivative<<<{"description": "Time derivative Kernel that acts on a coupled vector variable. Weak form: $(\\vec{\\psi}_i, \\frac{\\partial \\vec{v_h}}{\\partial t})$.", "href": "VectorCoupledTimeDerivative.html"}>>>
    variable<<<{"description": "The name of the variable that this residual object operates on"}>>> = v
    v<<<{"description": "Coupled vector variable"}>>> = u
  [../]
  [./diff_v]
    type = VectorDiffusion<<<{"description": "The Laplacian operator ($-\\nabla \\cdot \\nabla \\vec{u}$), with the weak form of $(\\nabla \\vec{\\phi_i}, \\nabla \\vec{u_h})$.", "href": "VectorDiffusion.html"}>>>
    variable<<<{"description": "The name of the variable that this residual object operates on"}>>> = v
  [../]

###########################################################
# This is a simple test of the VectorCoupledTimeDerivative kernel.
# The expected solution for the vector variable v is
# v_x(x) = 1/2 * (x^2 + x)
# v_y(x) = 1/2 * (x^2 + x)
###########################################################

[Mesh]
  type = GeneratedMesh
  nx = 5
  ny = 5
  dim = 2
[]

[Variables]
  [./u]
    family = LAGRANGE_VEC
  [../]
  [./v]
    family = LAGRANGE_VEC
  [../]
[]

[Kernels]
  [./time_u]
    type = VectorTimeDerivative
    variable = u
  [../]
  [./fn_u]
    type = VectorBodyForce
    variable = u
    function_x = 1
    function_y = 1
  [../]
  [./time_v]
    type = VectorCoupledTimeDerivative
    variable = v
    v = u
  [../]
  [./diff_v]
    type = VectorDiffusion
    variable = v
  [../]
[]

[BCs]
  [./left]
    type = VectorDirichletBC
    variable = v
    boundary = 'left'
    values = '0 0 0'
  [../]
  [./right]
    type = VectorDirichletBC
    variable = v
    boundary = 'right'
    values = '1 1 0'
  [../]
[]

[Executioner]
  type = Transient
  num_steps = 1
  solve_type = 'NEWTON'
[]

[Outputs]
  exodus = true
[]

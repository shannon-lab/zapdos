MoveNodesToSphere

Snap nodes to the surface of a sphere either during adaptivity or on execution

Moves all nodes from the supplied "boundary" or "block" to the surface of the sphere specified by the "center" and "radius" parameters.

This is performed by default during adaptivity to allow adaptive refinement of curved geometries. It can also be performed using regular execution schedules specified by the "execute_on" parameter.

    C++ Type:std::vector<SubdomainName>

    Unit:(no unit assumed)

    Controllable:No

    Description:List of blocks whose nodes are snapped to a given geometry

    Default:NONE

    C++ Type:ExecFlagEnum

    Unit:(no unit assumed)

    Options:NONE, INITIAL, LINEAR, NONLINEAR_CONVERGENCE, NONLINEAR, POSTCHECK, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, FINAL, CUSTOM

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed. For a description of each flag, see https://mooseframework.inl.gov/source/interfaces/SetupInterface.html.

    Description:Sphere radius

    Description:For the old and older state use projected material properties interpolated at the quadrature points. To set up projection use the ProjectedStatefulMaterialStorageAction.

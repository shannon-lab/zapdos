NodalNormalsCorner

Computes nodal normals at boundary corners.

The NodalNormalsCorner is added automatically by the AddNodalNormalsAction. This object is created for the corners specified in the "corner_boundary" parameter of the action. See the NodalNormals System for more information.

The corner normal is saved in the nodal_normal_x, nodal_normal_y and nodal_normal_z variables.

Input Parameters

  • boundaryThe list of boundary IDs from the mesh where this object applies

    C++ Type:std::vector<BoundaryName>

    Controllable:No

    Description:The list of boundary IDs from the mesh where this object applies

  • corner_boundaryNode set ID which contains the nodes that are in 'corners'.

    C++ Type:BoundaryName

    Controllable:No

    Description:Node set ID which contains the nodes that are in 'corners'.

ElementQualityChecker

Class to check the quality of each element using different metrics from libmesh.

Description

UserObject based on available libMesh quality metrics to check the quality of each element. It works for both uniform or mixed element types.

Example Input Syntax

[UserObjects<<<{"href": "../../syntax/UserObjects/index.html"}>>>]
  [./elem_quality_check]
    type = ElementQualityChecker<<<{"description": "Class to check the quality of each element using different metrics from libmesh.", "href": "ElementQualityChecker.html"}>>>
    metric_type<<<{"description": "Type of quality metric to be checked"}>>> = STRETCH
    failure_type<<<{"description": "The way how the failure of quality metric check should respond"}>>> = ERROR
    upper_bound<<<{"description": "The upper bound for provided metric type"}>>> = 1.0
    lower_bound<<<{"description": "The lower bound for provided metric type"}>>> = 0.5
  [../]
[]
(moose/test/tests/userobjects/element_quality_check/failure_error.i)

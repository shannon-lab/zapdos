NearestPointIntegralVariablePostprocessor

The domain is virtually divided into a number of subdomains according to the nearest points provided by users. And then the variable integral is taken over each individual subdomain separately.

warningwarning

Note that if "positions_object" is used to specify the nearest points, only the vector of initial Positions are used at this time. Updates to the 'positions' vector are not supported.

Compute element variable integrals for nearest-point based subdomains

ConvectiveFluxBC

Determines boundary values via the initial and final values, flux, and exposure duration

Description

The ConvectiveFluxBC boundary condition determines the value on a boundary based upon the initial and final values, the flux through the boundary and the duration of exposure.

Example Input Syntax

[BCs<<<{"href": "../../syntax/BCs/index.html"}>>>]
  active<<<{"description": "If specified only the blocks named will be visited and made active"}>>> = 'left right'

  [./left]
    type = DirichletBC<<<{"description": "Imposes the essential boundary condition $u=g$, where $g$ is a constant, controllable value.", "href": "DirichletBC.html"}>>>
    variable<<<{"description": "The name of the variable that this residual object operates on"}>>> = u
    boundary<<<{"description": "The list of boundary IDs from the mesh where this object applies"}>>> = 3
    value<<<{"description": "Value of the BC"}>>> = 0.0
  [../]

  [./right]
    type = ConvectiveFluxBC<<<{"description": "Determines boundary values via the initial and final values, flux, and exposure duration", "href": "ConvectiveFluxBC.html"}>>>
    variable<<<{"description": "The name of the variable that this residual object operates on"}>>> = u
    boundary<<<{"description": "The list of boundary IDs from the mesh where this object applies"}>>> = 1
    rate = 100
    initial = 10
    final = 20
    duration = 10
  [../]
[]

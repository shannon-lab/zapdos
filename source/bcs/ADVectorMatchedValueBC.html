ADVectorMatchedValueBC

Implements a ADVectorNodalBC which equates two different Variables' values on a specified boundary.

Description

ADVectorMatchedValueBC is a ADVectorNodalBC which applies to systems of two or more variables, and can be used to impose equality of two solutions along a given boundary. This class is appropriate for systems of partial differential equations (PDEs) of the form where is the domain, and is its boundary, , are the unknowns, , are forcing functions (which may depend on both and ), and and are given fluxes. The v parameter is used to specify the variable whose value is tied to . In the example below, the other variable's name happens to be v as well.

Example Input Syntax

  [./middle]
    type = ADVectorMatchedValueBC
    variable = v
    boundary = 'primary0_interface'
    v = u
  [../]
(moose/test/tests/interfacekernels/ad_coupled_vector_value/coupled.i)
